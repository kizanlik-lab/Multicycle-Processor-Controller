# Traffic Light Controller FSM
**Hacettepe University - ELE432 Advanced Digital Design (Homework 2)**

This repository contains the SystemVerilog implementation of an Multicycle Processor Controller.

## Project Overview
- Main Decoder Finite State Machine (FSM) design.
- A testbench module written for ModelSim/Questa that reads test vectors from a .tv file to verify the design.

## Files
- `controller.sv`: The main Controller design code.
- `tb_controller.sv`: The testbench code for simulation and verification.
- `tv_controller.tv`: The test scenarios read by the testbench.
